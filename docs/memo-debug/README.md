# メモアプリデバッグ プロジェクト

ichibou のメモ機能まわりの不具合・改善点を整理し、実装に落とすためのワークスペース。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `STATE.md` | **正本**。実装済み / 未実装 / 不要 の仕分け表。常にここを最新化する |
| `log-template.md` | 思考ログを書くときのひな形 |
| `log-NN.md` | 個別の思考ログ（時系列）。`log-01.md` から順に |

## 使い方

1. PC で書き溜めた思考ログを `log-01.md` 〜 `log-15.md` に貼り付ける
2. 1 件ずつ読みながら `STATE.md` の該当行を更新（実装済み / 未実装 / 不要 のいずれかに分類）
3. 「未実装」になった項目はそのまま着手リストとして使える
